# investments/
Portfolio data, investment playbook, per-position theses, and Yahoo Finance scripts.

**Always load `playbook.md` at the start of any investment session.** It contains the investor profile, all Yahoo Finance Python scripts, analysis framework, and risk rules.

| File | When to load |
|---|---|
| `playbook.md` | Load at the start of every investment or portfolio session — scripts, analysis framework, conventions |
| `portfolio.csv` | Load when fetching positions, calculating values, or reviewing account balances |
| `net-worth.csv` | Load when tracking total net worth across all accounts |
| `theses.md` | Load when asked about a specific holding's thesis, exit signal, or conviction level |
