Outland spousal sponsorship overview — wife (principal applicant) in India, Zee (sponsor) in Toronto.

---

## Stream

| Item | Detail |
|---|---|
| **Application type** | Outland spousal sponsorship (wife outside Canada) |
| **Sponsor** | Zee — Canadian PR or citizen in Toronto |
| **Principal applicant** | Wife — currently in India |
| **Processing** | Outland — typically faster than inland; wife can visit Canada while processing (visitor visa separate) |
| **Official guide** | [IMM 5289](https://www.canada.ca/en/immigration-refugees-citizenship/services/application/application-forms-guides/guide-5289-sponsor-your-spouse-common-law-partner-conjugal-partner-dependent-child-complete-guide.html) |
| **Document checklist** | [IMM 5533](https://www.canada.ca/en/immigration-refugees-citizenship/services/application/application-forms-guides/imm5533.html) — download current PDF from IRCC |

Always generate the application package via IRCC's online tool at [ircc spouse application](https://ircc.canada.ca/english/information/applications/spouse.asp) — country-specific requirements differ.

## Who fills what

| Role | Forms / actions |
|---|---|
| **Sponsor (Zee)** | IMM 1344 (PDF), IMM 5532 (PDF) — download, complete, digitally sign with wife |
| **Principal applicant (wife)** | IMM 0008, IMM 5669, IMM 5406, IMM 5562 (if applicable) — completed online in portal |
| **Both** | Upload supporting documents per IMM 5533 checklist; pay fees; wife e-signs full application |

## Outland vs inland (why outland)

| Factor | Outland (chosen) | Inland |
|---|---|---|
| Wife's location | India | Would need to be in Canada |
| Processing | Historically faster for India VAC stream | Longer; different queue |
| Visit during processing | Possible on visitor visa | Can apply for OWP if eligible |
| Appeal rights | Different if refused — confirm current rules | Different pathway |

---

## Critical rules

- Use **current** form versions from IRCC — outdated forms = returned application
- Non-English/French documents need **certified translation**
- Missing checklist items → include **detailed explanation letter** or IRCC may return as incomplete
- **Relationship evidence** is the most scrutinized section — see `spousal-pr-checklist.md`
- Do **not** store passport numbers, SIN, or scanned identity docs in this repo — track status only
