**Trading platform**: Wealthsimple (commission-free) — all trades executed here. Wealthsimple supports options trading in the non-registered account only (not TFSA/RRSP/FHSA).

**Research platform**: RBC Direct Investing — $10/trade so never used for execution, but includes free Morningstar reports (fair value estimates, moat ratings, star ratings, analyst commentary) for every stock.

**Why:** Use RBC exclusively as a research tool to pull Morningstar data, then execute on Wealthsimple.

**How to apply:** When the portfolio-review skill calls for Morningstar fair value or moat data, instruct the user to look it up on RBC Direct Investing (no subscription needed — it's included with the account). Do not recommend a Morningstar subscription.
